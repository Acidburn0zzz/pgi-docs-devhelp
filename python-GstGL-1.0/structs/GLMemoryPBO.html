Details

class GstGL.GLMemoryPBO

GstGL.GLMemoryPBO is created or wrapped through gst_gl_base_memory_alloc() with GstGL.GLVideoAllocationParams.

GstGL.GLMemoryPBO is created through gst_gl_memory_pbo_alloc() or system memory can be wrapped through gst_gl_memory_pbo_wrapped().

Data is uploaded or downloaded from the GPU as is necessary.
