static new()
Returns:a new Gdict.Defbox widget.
Return type:Gtk.Widget

Creates a new Gdict.Defbox widget. Use this widget to search for a word using a Gdict.Context, and to show the resulting definition(s). You must set a Gdict.Context for this widget using Gdict.Defbox.set_context ().
